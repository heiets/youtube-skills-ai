# Skills in AI tools — companion site

Static companion pages for the YouTube channel [Сашко пише код](https://www.youtube.com/@dev.sashko): an eight-chapter guide to **skills** (rules, AGENTS.md, Claude Skills, Cursor Rules, and related patterns) in AI coding tools.

## Live site

**https://youtube-skills-ai.vercel.app**

## Pages

| URL | Chapter |
|-----|---------|
| `/` or `/index.html` | Overview |
| `/what-are-skills.html` | What skills actually are |
| `/claude-skills.html` | Claude Skills |
| `/cursor-rules.html` | Cursor Rules |
| `/tool-comparison.html` | Tool comparison |
| `/create-skill.html` | How to create a skill |
| `/best-practices.html` | Best practices |
| `/pros-cons.html` | Pros & cons |
| `/conclusions.html` | Conclusions |
